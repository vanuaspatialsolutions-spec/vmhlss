MODELS (SQLAlchemy ORM Classes)
================================

1. app/models/__init__.py
   - Central import hub for all models
   - Exports all 7 model classes

2. app/models/user.py
   - User class for 'users' table
   - 10 columns with UUID PK
   - Unique email constraint
   - Role-based access control

3. app/models/dataset_slot.py
   - DatasetSlot class for 'dataset_slots' table
   - 11 columns, UUID PK
   - 14 slots: DS-01 through DS-14
   - ARRAY type for accepted_formats
   - JSONB type for required_attributes

4. app/models/dataset_upload.py
   - DatasetUpload class for 'dataset_uploads' table
   - 22 columns, UUID PK
   - GeoAlchemy2 POLYGON geometry
   - ARRAY[JSONB] type for fix_log
   - QA status tracking

5. app/models/knowledge_base.py
   - KnowledgeBaseRecord class for 'knowledge_base_records' table
   - 16 columns, UUID PK
   - JSONB types for metadata and tags
   - Full-text searchable content

6. app/models/analysis.py
   - Analysis class for 'analyses' table
   - 29 columns, UUID PK
   - GeoAlchemy2 MULTIPOLYGON geometries
   - Suitability assessment results storage
   - Processing status tracking

7. app/models/audit_log.py
   - AuditLog class for 'audit_log' table
   - 11 columns, BIGSERIAL PK
   - Immutable append-only design
   - INET type for IP addresses
   - No UPDATE/DELETE allowed

8. app/models/report.py
   - Report class for 'reports' table
   - 27 columns, UUID PK
   - GeoAlchemy2 MULTIPOLYGON geometry
   - Multiple export format support
   - Status and access control

9. migrations/__init__.py

10. migrations/alembic.ini
    - Alembic configuration file

    - Points to migrations directory

11. migrations/env.py
    - Alembic environment setup
    - Async SQLAlchemy support

12. migrations/versions/__init__.py

13. migrations/versions/001_initial_schema.py
    - Complete initial migration
    - Creates 9 tables:
      * users (7 core)
      * dataset_slots
      * dataset_uploads
      * knowledge_base_records
      * analyses
      * audit_log
      * reports
      * vanuatu_places (reference)
      * ahp_weights (reference)

    - Creates uuid-ossp extension
    - Creates audit_log immutability trigger
    - Full downgrade support
    - Strategic indexes on FK and frequently queried columns
